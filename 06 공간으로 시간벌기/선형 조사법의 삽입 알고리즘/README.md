### 선형 조사법의 삽입 알고리즘 | p.258

* key는 해시 함수에 적용해 해시 주소 id를 구할 킷값이에요.

* id 번째 테이블이 비어있지 않으면 계속 다음 위치를 검사해요.
* 비어있는 버킷을 찾으면 키를 저장해요.
